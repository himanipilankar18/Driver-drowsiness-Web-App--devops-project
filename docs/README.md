# Documentation Index

This folder contains hardware and integration documentation for the Driver Safety System.

## Files

- `HARDWARE_QUICKSTART.md` - Fast setup path for ESP32 hardware integration.
- `HARDWARE_INTEGRATION.md` - End-to-end integration details.
- `HARDWARE_CHECKLIST.md` - Deployment and debugging checklist.
- `HARDWARE_WIRING.md` - Wiring diagrams and pin mapping.
- `HARDWARE_SETUP_MULTISTATE.md` - Multi-state LED/buzzer setup guide.
- `HARDWARE_IMPLEMENTATION_SUMMARY.md` - Implementation status and summary.
- `ESP32_FLASH_GUIDE.md` - Firmware flashing instructions.

## Recommended Reading Order

1. `HARDWARE_QUICKSTART.md`
2. `HARDWARE_WIRING.md`
3. `ESP32_FLASH_GUIDE.md`
4. `HARDWARE_SETUP_MULTISTATE.md`
5. `HARDWARE_CHECKLIST.md`
